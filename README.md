# fifa-WC---project
This project builds an end-to-end machine learning pipeline to predict Win / Loss / Draw probabilities for FIFA World Cup 2026 group-stage matches. It ingests historical World Cup data (1930–2022), cleans and standardises country names, engineers custom features, merges live FIFA rankings, and trains multiple classifiers evaluated on a strict temporal (chronological) train-test split.

Best Model: Logistic Regression — 62.5% Accuracy | Macro F1: 0.59
